# Trello Image Guide

Use 4 to 5 images total. That is enough to make the board look complete without overloading every card.

## Recommended Images And Card Mapping

### 1. Convert POI shapefiles

Attach:

- `docs/trello_images/01_poi_indicator_counts.png`

Why:

- Shows that raw POI data was cleaned and classified into project indicators.

### 2. Build 500 m grid

Attach:

- `docs/trello_images/02_grid_sample.png`

Why:

- Shows that the spatial sampling grid was successfully generated.

### 3. Aggregate to H3

Attach:

- `docs/trello_images/03_h3_score_map.png`

Why:

- Best analytical image for the final scoring result.

### 4. Final reproducibility check

Attach:

- `docs/trello_images/04_top_hex_table.png`

Why:

- Shows the final scored outputs and top-performing hex cells in a compact summary.

### 5. Build static map shell

Attach:

- `docs/trello_images/05_web_overview.png`

Why:

- Shows the web app layout, control panel, and map container.

## Optional

You can also attach `docs/trello_images/06_web_interaction.png` to:

- `Implement required interactions`

Only use it if you think the screenshot looks clear enough on your screen.

## Do Not Attach Everywhere

Avoid putting the same image on multiple cards. One image per key card is enough.

## Best Upload Order

1. `Convert POI shapefiles` -> `01_poi_indicator_counts.png`
2. `Build 500 m grid` -> `02_grid_sample.png`
3. `Aggregate to H3` -> `03_h3_score_map.png`
4. `Build static map shell` -> `05_web_overview.png`
5. `Final reproducibility check` -> `04_top_hex_table.png`
