# Deploy Guide

Deploy the `web/` folder as a static site.

## Vercel

- Import or upload the `web/` folder.
- Framework preset: `Other`
- Root directory: `web`
- Build command: leave empty
- Output directory: leave empty

## GitHub Pages

- Push the project to GitHub.
- Publish the contents of the `web/` folder.
- If using a separate branch, copy only `web/` contents into that branch root.

## Required Public Files

- `index.html`
- `app.js`
- `styles.css`
- `data/hexes.json`
- `data/metadata.json`
